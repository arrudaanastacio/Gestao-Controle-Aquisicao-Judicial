Elenco de medicamentos

Planilha com as colunas Código do Item, Código Siafísico e Descrição. Itens novos são cadastrados, itens existentes são atualizados, e itens que saírem da lista são inativados (nunca excluídos) — com alerta se já tiverem histórico de compra.

Novas aquisições (solicitações mensais)

Planilha no layout original (uma aba por mês, ex.: AGOSTO-2026). Só linhas com algum campo de movimento preenchido são importadas; linhas totalmente vazias são ignoradas.

Estoque diário (Itens em Estoque UDTP)

Relatório de estoque no layout "Itens em Estoque UDTP". Cada importação é uma foto do dia — o histórico é mantido para acompanhar a evolução. A data é detectada pelo nome da aba; ajuste se necessário. Ao importar, os alertas de estoque são recalculados.

Configuração de alertas de estoque

Define a autonomia mínima (em meses) abaixo da qual um item é considerado com estoque baixo. Itens com estoque zero e demanda continuam sempre como ruptura (crítico).
